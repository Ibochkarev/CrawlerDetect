# Превью для ModStore и соцсетей

Материалы для публикации дополнения **CrawlerDetect** (определение ботов и защита форм от спама) на [modstore.pro](https://modstore.pro) и для превью в GitHub/соцсетях.

## Файлы

| Файл | Назначение |
|------|------------|
| **DESCRIPTION.md** | Развёрнутое описание возможностей — для ModStore и маркетинга |
| **logo-320x240.png** | 320×240 px — иконка/логотип для карточки в каталоге ModStore, аватар репозитория, соц-превью |
| **banner-1200x630.png** | 1200×630 px — баннер для страницы пакета на ModStore, Open Graph-превью (соцсети, мессенджеры) |
| **PackageLogFormatter.php** | Класс для красивого вывода логов install/upgrade/uninstall |
| **migration-log-template.md** | Шаблоны логов для установки, обновления, удаления пакетов |

## Использование

- **ModStore (личный кабинет):** загрузите логотип в карточку пакета; баннер — в описание или галерею.
- **GitHub:** укажите `logo-320x240.png` как изображение в описании репозитория или используйте баннер в соц-превью.
- **Соцсети:** `banner-1200x630.png` подходит для Open Graph (og:image); при уменьшении сохраняет читаемость.

Папка `_preview/` не входит в транспортный пакет компонента — только для репозитория и загрузки на ModStore.

## Изображения

- **logo-320x240.png** — логотип 320×240 px: иконка детекции ботов/защиты форм (ModStore, GitHub, соц-превью). Стиль: современный, лаконичный, узнаваемый в малом размере.
- **banner-1200x630.png** — баннер 1200×630 px: название «CrawlerDetect», подзаголовок «Защита форм от спама без CAPTCHA», нейтральный современный фон (ModStore, Open Graph). Текст читаемый при уменьшении.
